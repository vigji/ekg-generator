# CLAUDE.md

## Project

EKG Generator — produces realistic animated ECG/SpO2/ETCO2 monitor waveforms for clinical education. Outputs GIF or MP4 files.

## Key Commands

- **Install**: `pip install -e .`
- **Generate all presets**: `python scripts/demo.py`
- **Output**: animations go to `output/` directory
